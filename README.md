# What's for Dinner?

AI-powered meal planning for families. Tell the app about your kitchen, your family, and your preferences — get 3 personalized meal ideas in seconds.

**Live at [dinner-app.fly.dev](https://dinner-app.fly.dev)**

## What it does

- **4 meal flows**: Use what you have (pantry), hero ingredient, cuisine mood, or surprise me
- **AI suggestions**: Claude generates 3 meal ideas tailored to your family, cookware, and taste
- **Kid-friendly**: Every recipe includes a task for little helpers
- **Favorites**: Save recipes with a heart button, browse them later at `/favorites`

## Stack

- **Ruby 3.3** / **Rails 8.1** — server-rendered with Hotwire (Turbo + Stimulus)
- **BAML** — typed AI prompts targeting Claude Sonnet
- **Tailwind CSS** — styling via tailwindcss-rails
- **SQLite** — database (development and production)
- **Devise** — authentication
- **Fly.io** — deployment

## Setup

```bash
# Prerequisites: Ruby 3.3.7, bundler
# Recommended: mise (https://mise.jdx.dev)

git clone https://github.com/Gitzman/dinner-app.git
cd dinner-app
bundle install
bin/rails db:prepare

# Set your Anthropic API key for AI meal suggestions
echo "ANTHROPIC_API_KEY=sk-ant-..." > .env

# Start the server
bin/dev
```

Open [localhost:3000](http://localhost:3000), create an account, fill out your profile, and pick a flow.

## Environment variables

| Variable | Required | Description |
|----------|----------|-------------|
| `ANTHROPIC_API_KEY` | Yes | Claude API key for meal suggestions |
| `SECRET_KEY_BASE` | Production | Rails secret (auto-generated by Fly) |

## Project structure

```
app/
  controllers/
    meals_controller.rb      # Meal flows + favorite toggle
    favorites_controller.rb  # Saved recipes page
    profiles_controller.rb   # Family/preference setup
  models/
    user.rb                  # Devise auth + favorites association
    meal_suggestion.rb       # AI-generated meal ideas (JSON)
    favorite.rb              # Join table for saved recipes
  javascript/controllers/
    favorite_controller.js   # Stimulus: optimistic heart toggle
    loading_controller.js    # Stimulus: meal generation overlay
  services/
    meal_suggester.rb        # BAML client wrapper
baml_src/
  meals.baml                 # AI prompt definitions
  types.baml                 # UserProfile, MealIdea types
  clients.baml               # Claude client config
```
